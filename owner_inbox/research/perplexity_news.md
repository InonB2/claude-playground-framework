# Perplexity — Latest News (2026-05-09)

## Key Updates

- **Personal Computer for Mac — available to all Pro/Max subscribers (May 7, 2026)** — Completely redesigned native Mac app replaces previous macOS software. Agentic assistant that operates across local files, native apps, web, and Perplexity's secure servers. Invoked via Command Bar (press both Command keys). Was previously limited to $200/month Max plan; now extended to all Pro subscribers.
- **GPT-5.5 as default orchestration model for Computer (May 4, 2026)** — GPT-5.5 replaces previous model for Computer on Pro and Max plans. GPT Image 2 is now default for image generation within Computer. Focus: faster complex task execution.
- **Enterprise Computer features (May 4):**
  - Plan approval system — review task approach before execution begins
  - Inline file diffs — see exactly what Computer changed in files
  - Website publishing to persistent `*.pplx.app` URLs — no infrastructure setup required
  - Microsoft Teams integration
  - Snowflake and Databricks direct connectivity — warehouse data access without export
  - Workflows for repeatable tasks (market research, sales prep, slide creation)
  - Space Skills — package specialized capabilities across teams
- **Perplexity API becomes full-stack platform** — Four APIs: Agent API (multi-step workflows), Search API (real-time web retrieval), Embeddings API (retrieval at scale), Sandbox API (secure code execution, coming soon).
- **Perplexity Health (rolling out)** — Connects health data from medical records (1M+ providers), Fitbit, Google Fit, etc. Available to Pro/Max subscribers.
- **Personal CFO feature (April 17)** — Connects to bank accounts via Plaid for spending tracking and portfolio visualization.
- **"Try It On" feature** — Virtual avatar to preview apparel items on a personalized model.
- **Memory + Model Council** — Extended memory now works with Model Council, pulling personalized context for each AI model in the council.
- **Real-time financial newswire** — AI-generated market synthesis with Watchlist tab; tracks individual equities and market movements.
- **Perplexity valued at $21B** — Answer engine positioning, ongoing legal battles around content attribution.
