# Google Gemini — Latest News (2026-05-09)

## Key Updates

- **Gemini 3.1 rolling out in Google Home (May 5, 2026)** — Spring 2026 update brings Gemini 3.1 to Home devices as the "most advanced for home use." Multi-step voice commands now handled in a single utterance (e.g., "add milk to the list and move the dentist appointment to Thursday").
- **Gemini Agent ("Remy") announced as 24/7 digital partner** — Autonomous agent that can take web actions, communicate with others, share documents, and make purchases. Tracks tasks by status (completed/in-progress/pending/scheduled) in a dashboard. Currently experimental, available to Google AI Ultra subscribers only.
- **Workspace Intelligence launched** — Gives Workspace AI real-time context from Gmail, Chat, Calendar, and Drive. Generative tasks can use live work data without re-prompting. Native file creation (Docs, PDFs, Word, Excel, CSV, Markdown) from chat.
- **Gemini app major redesign in testing** — Pulsating gradient background, centered greeting, unified tools list (Deep Research, Canvas consolidated), pill-shaped prompt box. Visual overhaul signals consumer push.
- **Automotive integration** — Gemini rolling into millions of vehicles for natural voice requests. Local business SEO implications flagged by marketers.
- **Google I/O 2026 confirmed May 19-20** — Keynote at 10 AM PT. Expected: Gemini 3.1 Ultra (frontier model with Astra-level real-time audio/video in production API), Android 17, Aluminium OS debut, Android XR hardware, possible "Omni" video model brand replacing Veo.
- **Gemini API changelog** — Ongoing updates at ai.google.dev; Gemini 3.1 Ultra expected to bring real-time audio/video understanding to production API access.
