# OpenAI Codex — Latest News (2026-05-09)

## Key Updates

- **GPT-5.5 Instant becomes default ChatGPT model (May 5, 2026)** — 52.5% fewer hallucinations than GPT-5.3 Instant on high-stakes prompts (medicine, law, finance). Tighter responses, better personalization using past conversations, files, and Gmail integration.
- **GPT-5.5 also becomes Perplexity's default orchestration model** — Perplexity adopted GPT-5.5 for Computer on Pro/Max plans (see Perplexity file), signaling OpenAI's model licensing strategy broadening.
- **3 new real-time audio models (May 7, 2026):**
  - **GPT-Realtime-2** — Reasoning comparable to GPT-5, 128K context window, adjustable reasoning levels
  - **GPT-Realtime-Translate** — 70+ input languages, 13 output languages for live multilingual conversation
  - **GPT-Realtime-Whisper** — Low-latency streaming transcription for live apps
  - Pricing: Realtime-2 at $32/1M input tokens; Translate at $0.034/min; Whisper at $0.017/min
- **ChatGPT for Excel and Google Sheets — globally available (May 5)** — Formula building, data cleanup, multi-tab analysis. Available on Free, Plus, Pro, Business, Enterprise tiers.
- **ChatGPT Business admin console** — New Analytics and Agents sections; consolidated view of adoption, usage patterns, and workspace agent management.
- **Self-serve Ads Manager beta (May 5)** — Cost-per-click bidding alongside CPM. Conversion tracking with privacy-preserving aggregated insights.
- **Advanced Account Security (April 30)** — Passkey/security key requirement, password login disabled for enrolled accounts, automatic training exclusion for sensitive content, extends to Codex accounts.
- **OpenAI Codex (the agent) context:** 2M+ weekly active users as of March 2026. Positioned as enterprise agent platform beyond coding. Apple deepened Xcode 26.3 integration. GitHub Agent HQ includes Codex in public preview (alongside Claude and Copilot). GPT-5.4 for Codex released March 5, 2026.
