## How to use it

1. **Print the marker** — click "Print Marker Page" on the demo, or print/display the Hiro marker image:
   `https://raw.githack.com/AR-js-org/AR.js/master/data/images/hiro.png`

2. **Open the URL on your phone** (iOS Safari or Android Chrome)

3. **Tap "Launch AR Camera"** and allow camera access

4. **Point phone at the Hiro marker** (~30–60 cm away)

5. **5-step HVAC repair instructions** float above the marker in AR

## What it demonstrates

- Zero-friction investor demo — shareable URL, no app install
- Core BuildARPro loop: physical marker trigger → digital AR overlay → sequential guided steps
- Branded UI (dark #0D0D0D, orange #F97316) with printable marker instructions on the same page
- Production extension path: swap Hiro for custom image targets, plug in backend CMS for instructions
