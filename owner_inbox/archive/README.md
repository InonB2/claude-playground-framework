# Owner Inbox — Archive

Completed and approved work lives here, ordered chronologically.

**Naming convention:** `YYYY-MM-DD_[agent]_[brief-description].md`  
**Example:** `2026-04-27_Cole_linkedin-post-draft.md`

Agents move work here after the Owner approves it from `owner_inbox/`. The Owner may also manually archive items. Andy scans this folder to find prior work when context is needed.
