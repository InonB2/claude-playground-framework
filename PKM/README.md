# PKM — Personal Knowledge Management

**Owner:** Inon Baasov  
**Purpose:** Personal insights, learnings, and knowledge — separate from team/business knowledge (BKM).

This folder is the Owner's private knowledge base. Agents may write here when the Owner instructs them to capture personal insights, but this folder is NOT used for team operational knowledge (that lives in `/BKM/`).

## Sub-folders

| Folder | Purpose |
|--------|---------|
| `daily_journal/` | Daily journal entries, reflections, observations |
| `insights/` | Distilled lessons, frameworks, patterns the Owner finds valuable |
| `references/` | Clippings, links, summaries of external content |

## How to add a journal entry

Tell Andy: _"Create a journal entry for today about [topic]."_  
Andy will delegate to **Cole** (copywriter) to write and file it in `daily_journal/YYYY-MM-DD.md`.

## Separation from BKM

- **BKM** = how the team operates (SOPs, processes, agent knowledge)
- **PKM** = what the Owner knows, thinks, and learns (personal)
